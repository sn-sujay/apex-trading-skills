# APEX Forex Paper Trade Engine

## Purpose
Paper trading for forex pairs:
- EUR/USD
- GBP/USD
- USD/JPY
- USD/INR

## Risk Parameters
- Max 2% daily loss
- Half-Kelly position sizing
- Tight stops on news events

## Note
Requires forex broker API (e.g., OANDA, broker with MT4/MT5).
